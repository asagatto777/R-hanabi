# R-hanabi
This is a project for PTB2 Class.   
筑波大学生物学類の科目「理論生物学の基礎Ⅱ」の課題「プログラミングで一発芸」のソースコードです。  
## Dependencies
- [R](https://www.r-project.org/)
## How to use
```source("script.R")```でスクリプトを実行すると夜空に花火が打ち上がります。  
1回実行後にコンソールで```uchiage()```と入力すると前に打ち上げた花火を残したまま打ち上げられます。
